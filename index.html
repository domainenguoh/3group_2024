CodeGenCrusaders

Unveiling the Impact of Cognitive Biases on Large Language Models

What is CodeGenCrusaders?

CodeGenCrusaders is a project exploring how large language models (LLMs) are influenced by cognitive biases in prompts. The project is conducted within the framework of a seminar course titled Software Engineering in the Age of AI at the Haifa University IS Department.
  • Analyzing cognitive biases
  • Understanding LLM behavior
  • Improving prompt design
  • Mitigating bias impact
Easily extensible for various research needs.

Documentation

The project involves various scripts, models, and datasets. Detailed documentation and usage instructions can be found in the respective files within the repository. Here is a brief overview of the key components:

  • Data Processing: Scripts to preprocess and handle data
  • Model Training: Code for training the LLMs with different prompt biases
  • Evaluation: Methods for assessing the impact of biases

For specific details, refer to the documentation within each directory of the project.
